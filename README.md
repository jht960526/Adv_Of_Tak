## 게임 설명
*게임 컨셉
```
산기대에 입학한 신입생 택.
기쁜 마음으로 수업을 들어갔지만 첫 시간부터 강렬한 교수님의 한마디
'프로그래밍 못하는 학생은 졸업시키지 말자'
신입생 정홍택의 졸업을 위한 생존기

```

*게임 설명
```
장르: 2d 횡스크롤 캐쥬얼 게임

```
